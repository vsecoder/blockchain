## Features

- Create a new blockchain
- Add a new transaction (send coins, NFTs)
- Check if the blockchain is valid
- Create NFTs
- Proof of work

## Installation

```bash
pip install requirements.txt
```

## Usage

### Edit the `config.toml` file from `example.toml`

```toml
key = "my_secret_key"
```

### Run the server

```bash
python3 -m app
```
